# เช็คเลขบัตรประจำตัวประชาชน
## Thai Citizen ID validator

## Installation
```
bower install th-id-validator
```

## Usage
HTML

```
<script src="../th-id-validator.min.js"></script>

```

Javascript

```

var result = validateThaiCitizenID(‘<ID>’);

```

## Development
### Testing
```
gulp
```

### Build
```
gulp build
```
